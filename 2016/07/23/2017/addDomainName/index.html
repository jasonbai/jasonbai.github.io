绑定个人域名到Github Page（Hexo版）

发表于 2016-07-23

在依照“使用 Hexo 为自己在 Github 上建一个静态 Blog 站点” 后，进一步优化站点。

目前做了以下升级：

  • 通过修改theme的CSS，调整了部分样式
  • 绑定个人域名，替换Github提供的二级域名

本文简单介绍一下绑定二级域名的过程。

注册个人域名，且修改CNAME指向

在域名服务提供商注册域名后，修改记录类型CNAME,记录值为现有的Github的二级域名（例如我个人的jasonbai.github.io）

添加CNAME文件

在~/source/ 文件下添加文件名为CNAME文件，文件内容添加个人域名地址（例如我个人的www.jasonbai.com）

上传CNAME


hexo generate


hexo deploy
