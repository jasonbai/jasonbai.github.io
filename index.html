Markdown使用MathJax引擎插入数据公式

发表于 2017-04-07

一、目标：

想在博客里写一些数学公式，网上搜罗一番材料，最终选择使用MathJax引擎。

MathJax是一个JavaScript引擎，用来显示网络上的数学公式。它可以工作于所有流行的浏览器上。

二、步骤：

2.1 在Markdown编辑器里添加一下代码，调用MathJax引擎。

2.2 代码说明

使用Tex写公式。$$公式$$表示行间公式，本来Tex中使用\(公式\)表示行内公式，但因为Markdown中\是转义字符，所以在Markdown中输入行内公式使用\\(公式\\)，如下代码：


$$x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$$
\(x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}\)

行间显示
$$x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$$

行内显示
\(x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}\)

本地Drools Workbench环境搭建及安装配置

发表于 2017-03-30

概述

上周出差，发现不少互联网公司开始采用Drools平台来搭建风控决策引擎，计划在本机环境测试搭建Java平台开源决策引擎Drools，并进行系列的文章来跟踪记录整个学习过程。

Drools介绍

Drools（JBoss Rules ）具有一个易于访问企业策略、易于调整以及易于管理的开源业务规则引擎，符合业内标准，速度快、效率高。业务分析师或审核人员可以利用它轻松查看业务规则，从而检验是否已编码的规则执行了所需的业务规则。

官方网站：http://www.drools.org/

环境设置

我的本机系统环境 MacOS 10.11.6，本机运行Drools Workbench需要有Tomcat和MySQL，个人电脑没有配置，需要从头搭建环境。

安装Tomcat

安装Tomcat，参考文章

  1. 启动服务：打开终端， cd /Users/apple/Library/Tomcat/bin
  2. 再输入： ./startup.sh
  3. 网址输入 http://localhost:8080/
  4. 关闭服务：终端输入： ./shutdown.sh

安装MySQL

  1. 官网下载mysql安装包mysql-5.7.13-osx10.11-x86_64.bmg安装

  2. 安装完成后终端输入：

mysql –version
—-显示版本号说明正常，若显示command not found，在终端输入如下，”/usr/local/mysql/bin/mysql”为mysql默认安装路径：


$ cd /usr/local/bin/

$ sudo ln -fs /usr/local/mysql/bin/mysql mysql

  1. 关闭mysql服务：

sudo /usr/local/mysql/support-files/mysql.server stop

  1. 配置root账号的密码，默认没有配置，

—–在终端内输入（开启安全模式启动mysql）：

sudo /usr/local/mysql/bin/mysqld_safe –skip-grant-tables

—–修改密码，终端输入（逐行输入，必须，“**“为设置的密码）：

  1. 配置完成后验证，在终端输入：

mysql -u root -p
—-提示输入密码，进入mysql说明成功！

mysql>

  1. mysql服务启动和关闭可以在系统偏好设置里面操作，终端命令如下：

    $ sudo /usr/local/mysql/support-files/mysql.server start
    $ sudo /usr/local/mysql/support-files/mysql.server stop

发表于 2017-02-08

今天开始学习描述性统计学入门课程，随堂进行笔记记录。

1.什么是统计学？

统计学是数学的一个重要领域，用于分析、解释和预测数据的结果。描述统计学将让你学会描述数据的基本概念。这是为那些对数据科学、经济学、心理学、机器学习、体育分析学以及其它任何领域感兴趣的人开设的意义重大的入门课程。本课程提供中文版本。

2.什么是描述性统计？

描述性统计是指将调查样本中的包含的大量数据资料进行整理、概况和计算。是推断性统计的基础。

2.1 主要运用工具

集中量数，如平均数、中位数、众数、几何平均数、调和平均数。

变异量数，如全距、平均差、标准差、相对差、四分差。

常见的描述性统计方法可分为三类：

  • 用数据的统计量来描述，如：均值、标准差等；
  • 用图示技术来描述，如：直方图、散布图、趋势图、排列图、条形图和饼分图等；
  • 用文字语言分析和描述，如：统计分析表、分层、因果图、亲和图和流程图等。

便携Wiki软件TiddlyWiki简明使用手册

发表于 2016-07-21

由于需要多个文档进行管理，且需要及时更新并公布给使用者。寻找到了TiddlyWiki这款软件。

一个基于 JavaScript 全交互维基。它可以作为浏览器中的单一HTML文件,高度可定制.

满足了目前的需求

  • 可以快速传播，U盘拷贝，网络传输，无需搭建环境
  • 高度定制，便捷的文档管理及索引
  • 支持插件，简单的WikiText语言
  • 支持中文，不需要任何特殊知识或工具

1.安装

中文官网：http://www.tiddlywiki.cn/

依照官网的GettingStarted进行,下载保存空的TiddlyWiki。

Tips:

  • 最好采用Chrome下载保存，Safari有可能会卡死
  • 不要更改扩展名

2.设置

在下载后，直接开个文件，进行设置。如果没有中文语言包，在设置中找到插件标签，进行插件安装。

TiddlyWiki支持Markdown插件，经过试用编辑起来bug较多，建议试用WikiText进行文档输入编辑。

WikiText语法

WikiText（维基语法）是一种简洁的、富于表现力的、支持广泛的文本格式、超文本和交互功能的输入方法。它可以让你专注于写作而不是提供复杂的用户界面。其设计亲近 MarkDown语法 的用户，但更关注链接和互动功能。

2.1 WikiText 常用语法

块引用


<<<
This is a block quoted paragraph
written in English
<<<

显示为

This is a block quoted paragraph
written in English

标题


! This is a level 1 heading


!! This is a level 2 heading


!!! This is a level 3 heading

前置输入!, ‘！’越多，标题字体越小

列表

列表控制由#和*进行有序列表、无序列表设置

3.保存和传输

TiddlyWiki在页面点击保存，在浏览器上保存后自行命名保存。注意采用相同命名后会替换原有版本。建议在为了版本管理，采用命名升级的方式。单一文件可以通过u盘拷贝复制，也可以通过网络传输。

4.上传展示

TiddlyWiki提供http://tiddlyspot.com 站点服务，可以将自己的wiki上传展示，个人大概看了这个站点，速度较慢，体验一般，并没有深入使用。

另外一种方式是丢在GitHub上面，如果有更新可以随时通过git的方式进行上传，但注意上传Github后wiki为开放状态，注意保护个人隐私。
将文件名命名为index.html ，使用github page服务进行展示的教程在这里：Creating Project Pages manually
