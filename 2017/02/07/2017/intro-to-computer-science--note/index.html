计算机科学导论完结笔记

发表于 2017-02-07

在2017Plan里，计划夯实计算机基础知识，这部分知识其实前期学习很多，但不是科班出身，没有系统学习。
这次算是一次系统的学习，对算法有了更深入的认识。对计算机的基本理念，利用编程解决现实问题做了很好的铺垫。

课程主要框架

  1. 如何重复
  2. 如何解决问题
  3. 如何管理数据
  4. 响应查询
  5. 程序如何运行

课程进度

在Udacity上建议学习时间是72h，3个月时间，我大概用了30h小时的样子，完成课程，但没有做到每一道练习题都学习到位。

遗留问题

还是有一些概念不太能透彻理解

  1. 哈希函数
  2. 递归与迭代

没有在不懂的地方，长期停留，边走边理解。
